# TS3-Tournament-Interface

## How to Build and Run

```sh
docker build -t my-php-app .
docker run -d --name my-running-app my-php-app
```

## How to Configure
To change the panle name and logo
```json
    "panelinfo":[
        {
            "pagename":"Leagueops - Match Manager",
            "pagetheme":"dark",
            "pagelogo":"leagueops.png"
        }
```
